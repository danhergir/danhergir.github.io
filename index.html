Work Experience

Data Science Specialist - Dataplicada (Aug 2020 - Present)

As the Chief Technology Officer (CTO) and Data Science Specialist at Dataplicada, I led the development of feedbacker.cx, a comprehensive platform designed to enable businesses to harness the power of data for strategic decision-making. The platform was built from scratch using JavaScript, React.js, Laravel, and PHP, and it was deployed on Linux-based servers to ensure high performance and reliability.

I also engineered an Extract, Transform, Load (ETL) system to capture and process customer feedback from more than 100 third-party businesses. This system utilized secure TCP/IP protocols to manage feedback transfer, ensuring data integrity and security. Additionally, I implemented a Retrieval-Augmented Generation (RAG) agent using advanced language models such as GPT-4 and Llama3, which extracted valuable insights from this feedback, helping companies optimize their R&D, Marketing & Sales, Production, HR, and Finance operations.

Software Engineer Intern - Tesla, Inc. (Sep 2022 - Dec 2022)

During my time at Tesla, I worked on the Powerwall production line, where I contributed to enhancing manufacturing efficiency by developing and modifying line layouts, improving material flow, reducing waste, and improving ergonomics. By utilizing Lean Manufacturing best practices, I was able to optimize operations, ultimately reducing adhesive dispense waste by 20%.

I also played a critical role in transitioning products from concept to pilot line production and into full-scale mass manufacturing. This involved improving the assembly line station logic to automate the generation of distinct serial numbers for multiple Powerwall specifications. Moreover, I integrated a Kubernetes architecture with GitHub Actions to create a CI/CD pipeline using Jenkins and Docker, ensuring a robust and automated deployment process.

Full Stack Web Developer - Consulting (Nov 2018 - Sep 2021)

As a Full Stack Web Developer, I collaborated with a team of developers to design and implement a virtual reality (VR) experience for Toyota Chile. This project was a strategic promotional initiative for a new vehicle and utilized cutting-edge technology, including JavaScript and KRpano, to deliver an immersive VR experience to customers. The success of this project provided Toyota with a novel way to engage with their target audience.

Additionally, I was responsible for designing and implementing a custom network protocol using C++ for real-time communication between embedded systems and a central server in an IoT project. The protocol was optimized for low-latency, high-throughput data transmission, ensuring reliable communication even under varying network conditions. In another key project, I integrated the Ford Argentina API into a customer relationship management (CRM) system used by sales personnel across multiple dealerships. This integration allowed for efficient tracking of vehicle orders, providing dealerships with critical insights into customer orders and streamlining the sales process using Laravel and PHP.
